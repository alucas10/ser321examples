You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /comb -- returns the number of combations with values n and k. n cannot be greater than 13. Example: /comb?n=13&k=7
  • /char -- returns the integer value of the first character in the parameter. Example: /char?ch=a
